R7RS index

R7RS index allows searching for R7RS-small and R7RS-large (work in progress) procedures, syntax and constants through types, tags, and names. Please see user guide for details

Handful examples

  • What's the syntax for `do` again?
    Query by phrase "do"
  • What does (scheme set) library has to offer?
    Query by library "(scheme set)"
  • I have a list, how can I functionally transform it?
    Query by tag "pure", parameter type "list?", return type "list?"
